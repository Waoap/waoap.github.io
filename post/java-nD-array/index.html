1. 声明、创建以及初始化

类似于一维数组。

在 Java 中，可以认为二维数组 变量 是一个存储着 存储着一维数组的变量 的 变量 。

Java 声明实现的效果

double[][] balances = new double[10][6];

不同于在 C++ 中的

double balances[10][6];
double (*balances)[6] = new doule[10][6];

而是分配了一个包含 10 个指针的数组，然后为这个指针数组的每一个元素填充了一个包含 6 个值的 double 数组：

double** balances = new double*[10];
for (i = 0; i< 10; i++)
    balances[i] = new double[6];

上面的比较，有助于我们理解二维乃至多维数组。

2. 获取二维数组的长度

二维数组实际上是一个数组，它的每个元素都是一个一维数组。数组 x 的长度是数组中元素的个数，可以用 x.length 获取该值。

元素 x[0] , x[1] 等也是数组。可以使用 x[0].length , x[1].length 等来获取它们的长度。

6. 多维数组

二维数组由一个存储着一维数组的数组组成，而一个三维数组可以认为是由一个存储着二维数组的数组所组成。

可以对二维数组变量的声明以及二维数组的创建方法进行推广，用于声明 >= 3 的 n 维数组变量和创建 n 维数组。
