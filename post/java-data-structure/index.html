Java 数据结构

2022-01-18 #Java

1、变量与常量；
2、基本数据类型；
3、类型转换；
4、编码、字符集。

1、变量与常量

变量：
变量用于表示特定类型的数据。
重点内容包括声明[1]、初始化（赋值）[2]、使用[3]、使用范围、在内存中的地址；

常量：
常值（直接量） ，如 "a" 、123 等；
命名常量 ，如由 final 关键字修饰的常量；

2、数据类型

基本数据类型： 整数类型、浮点数类型、字符类型、布尔类型；
引用数据类型： 类、接口、数组；

3、类型转换

byte、short、char -> int -> long -> float -> double
左 --> 右： 自动（隐式）转换；
左 <-- 右： 强制（显式）转换。

4、编码、字符集

常见：

  • 通用：ASCII、Unicode（UTF-8为主）；
  • 中国：GB2312、GBK；
  • 欧洲：IOS8859-1;
  • …… ……

  1. Java 是强类型语言，声明变量时必须定义类型； ↩︎

  2. ...(x = 1); 等价于 x = 1; ...(x);，这被称为赋值表达式； ↩︎

  3. 变量在使用前必须被声明和初始化； ↩︎
