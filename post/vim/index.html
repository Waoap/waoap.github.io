VIM

2021-12-26 Vim Coding Tools

介绍🎉

VIM，程序员必备工具，一种丢掉鼠标的优雅生活方式💕。

这里贴一段维基里百科里对VIM的介绍：

Vim (/vɪm/; a contraction of Vi IMproved) is a free and open-source, screen-based text editor program for Unix. It is an improved clone of Bill Joy's vi.

这里是VIM的官网链接以及GitHub仓库主页。

使用及建议👻

使用：在VSCode、Visual Studio、Idea等IDE中安装Vim插件。
建议：将以下快捷键交由IDE控制。（注：这部分内容将在我摸索使用VIM的过程中随之变更。）

快捷键 作用
ctrl-a 全选
ctrl-c 复制
ctrl-x 剪切
ctrl-v 粘贴
ctrl-n 新建文件
ctrl-o 打开文件
ctrl-f 查找
ctrl-r / ctrl-h 替换

学习计划📚

  1. 通过观看VIM的相关教程和VIM的官方文档来了解、学习VIM的快捷键，并做好笔记。
  2. 在任何时候都尝试使用VIM来操作，提升自己操作的熟练度。（多敲代码，不仅能提升编程的实操能力，还能熟悉VIM，岂不美哉！😍😘）
