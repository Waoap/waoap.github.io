1. 数组的声明、创建、初始化以及访问

在声明数组 变量 时，需指出数组类型和数组变量的名字，如：

int[] a;

不过这只声明了变量 a ，没有将 a 初始化为一个真正的数组。应使用 new 操作符创建数组：

int[] a = new int[100]; // arrayType[] arrayVarName  = new arrayType[arraySize]

或者使用 数组初始化语法 ，不需要使用 new 操作符，甚至不用指定长度，形如：

int[] smallPrimes = { 2, 3, 5, 7, 11, 13 }; // arrayType[] arraryVarName = { value1, value2, ... , valuek };

其中最后一个值后面允许有逗号，如果你要不断为数组增加值，这会很方便，如：

String authors = {
    "James Gosling",
    "Bill Joy",
    "Guy Steele",
    // add more names here and put a comma after each name
}

而后可以使用 arrayVarName[index] 来访问数组中下标为 index 的元素。

还可以声明一个 匿名数组 ：

new int[] { 17, 19, 23, 29, 31, 37 };

注：

  1. 一个数组变量实际上存储的是 指向数组的引用 。严格地讲，一个 数组变量（array variable） 和一个 数组（array） 是不同的，但多数情况下它们的差别是可以忽略的。因此，为了方便，通常我们说某数组变量 mArray 是一个数组，而不用说：mArray 是一个指向一个含有 n 个某类型元素的数组的引用变量。

  2. 在 Java 中，允许有长度为 0 的数组，如：

    new int[0];
new double[] {};

    注意，长度为 0 的数组与 null 并不相同，当我们调用 Java 程序而未给它传任何参数的时候，main 方法的参数 args 便是一个长度为 0 的数组，详见 Java 基本概念—— main 方法。

2. 数组的大小和默认值

使用 arrayVarName.length 来获取数组的大小 。

整型数组的默认值为 0（char 数组为 '\u0000' ，boolean 数组为 false ，对象数组为 null ），表示这些元素（还）未存放任何对象。

4. 将数组传递给方法、从方法中返回数组

格式样例：

public static int[] x(int[] num) {}

将一个数组（实际是数组变量）传递给方法时，数组的引用 被传给方法。

Java 使用 按值传递（pass by value） 的方式将实参传递给方法：

  1. 将基本数据类型变量作为参数，传递的（形参得到的）是实参变量的值

  2. 将引用数据类型变量作为参数，传递的（形参得到的）也是实参变量的值（但对于引用数据类型的变量来说，它的值是一个引用）

所以，将数组传给方法之后，如果改变方法中的数组，将会看到方法外的数组也变化了。

当从方法中返回一个数组时，得到的返回值也为 数组的引用 。

注： 数组在 Java 中也是 对象 ，JVM 将对象存储在一个称作 堆（heap） 的内存区域中，堆用于动态内存分配（相对的是 方法 存储在 栈 中 ）。

5. 可变长参数列表

具有 同样类型的、可变长度的 参数可以传递给方法，格式如下：

public static int[] x(int... num) {}

其将被 当作数组来对待 。

7. 数组的排序

选择排序、冒泡排序、插入排序等。

可以使用 Arrays 类中的 sort 方法，这个方法使用了优化的快速排序（QuickSort）。

8. Arrays 类

java.util.Arrays 类包括各种各样的静态方法，用于实现数组的排序和査找、数组的比较和填充数组元素，以及返回数组的字符串表示。

  1. 可以使用 sort 或者 parallelSort 方法对整个数组或部分数组进行排序

  2. 可以采用二分査找法（binary Search 方法）在数组中査找关键字

  3. 可以采用 equals 方法检测两个数组是否相等

  4. 可以使用 fill 方法填充整个数组或部分数组

  5. 还可以使用 toString 方法来返回一个字符串，该字符串代表了数组中的所有元素
