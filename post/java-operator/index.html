1. 运算符类型

  2. 赋值运算符（=）

  3. 关系运算符（> 、< 、>= 、<= 、==）

  4. 逻辑运算符（&& 、|| 、!）

1.6 位运算符

  1. &（按位与）

  2. |（按位或）

  3. ^（按位异或）

  4. ~（按位取反）

  5. <<（左移，0 填充低位）：

    用途：用于快速实现乘法，如 4 * 8 ，即 2^3 ，等价于 4 << 3 。

  6. >>（有符号右移，符号位填充高位 ）：

  7. >>>（无符号右移，0 填充高位）：类似 >>

位移运算符的右操作数要模 32 （除非左操作符数是 long 类型，这时要对右操作数模 64）。因为 1 << 35 的结果和 1 << 3 相等。
