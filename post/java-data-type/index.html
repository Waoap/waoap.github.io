2022-01-18 Java

1、变量与常量；
2、基本数据类型；

1、变量与常量

常量：字面常量，如 "a" 、123 等；
变量：如 int a = 1; 等，重点内容包括声明[1]、赋值、使用[2]、所处内存；

2、基本数据类型

包括：整数类型、浮点数类型、字符类型、布尔类型；

  1. Java 是强类型语言，声明变量时必须定义类型； ↩︎

  2. 使用前必须赋值（初始化）； ↩︎
