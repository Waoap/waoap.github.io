Java 数据类型*

2022-01-18 #Java

1、变量与常量；
2、基本数据类型；
3、类型转换；
4、编码、字符集；

1、变量与常量

常量：字面常量，如 "a" 、123 等；
变量：如 int a = 1; 等，重点内容包括声明[1]、赋值、使用[2]、所处内存；

2、数据类型

基本数据类型包括：整数类型、浮点数类型、字符类型、布尔类型；
引用数据类型包括：类、接口、数组；

3、类型转换

byte、short、char -> int -> long -> float -> double
--> 自动转换 || <-- 强制转换

4、编码、字符集

常见：

  • 世界：ASCII、Unicode（UTF-8为主）；
  • 中国：GB2312、GBK；
  • 欧洲：IOS8859-1;

  1. Java 是强类型语言，声明变量时必须定义类型； ↩︎

  2. 使用前必须赋值（初始化）； ↩︎
